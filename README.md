# BlueMath: 高中数学竞赛一试讲义
本项目主要收集整理不同竞赛教材中的题目，目前是我平时上竞赛课的讲义，当然也可改制成beamer课件，或是作为高中课内教学的一个备用拓展题库。难度定位在一试水平，有关课内教学的教案可以参考我的另一个项目[RedMath](https://github.com/yzdame/RedMath)，二试水平的内容将会被收集到另一个项目PurpleMath（仍在计划）中，敬请期待！
